# Telegram Bot – Automated Messaging & Contact Management

A Python-based Telegram bot built to automate bulk messaging workflows and manage contact lists via CSV, reducing manual communication effort.

## Features
- **Bulk Messaging** – Send automated messages to multiple users from a CSV contact list
- **CSV Import/Export** – Load contacts and export results for reporting
- **Modular Architecture** – Clean separation of bot logic, features, and utilities
